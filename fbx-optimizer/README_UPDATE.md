# 🔄 如何更新到最新版本

## 方法一：使用更新腳本（推薦）

### 在專案根目錄
雙擊執行 `更新到最新版本.bat`

### 在 fbx-optimizer 資料夾內
雙擊執行 `更新到最新版本.bat`

腳本會自動執行以下操作：
1. ✅ 從 GitHub 拉取最新代碼
2. ✅ 更新所有依賴套件
3. ✅ 檢查並報告更新狀態

---

## 方法二：手動更新

### 1. 拉取最新代碼
```bash
git pull origin main
```

### 2. 更新依賴套件
```bash
npm install
```

### 3. 啟動應用程式
執行 `一鍵啟動.bat` 或：
```bash
npm run dev
```

---

## 💡 常見問題

### Q: 更新失敗怎麼辦？
A: 請檢查：
- 網路連線是否正常
- 是否有未提交的本地修改（使用 `git status` 檢查）
- GitHub 權限是否正確

### Q: 出現代碼衝突？
A: 如果您有本地修改，請先：
```bash
git stash          # 暫存本地修改
git pull           # 拉取最新代碼
git stash pop      # 恢復本地修改
```

### Q: 如何查看更新內容？
A: 使用以下命令查看更新日誌：
```bash
git log --oneline -10
```
